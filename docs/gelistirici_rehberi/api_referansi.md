# API Referansı (İskelet)

Bu belge, çekirdek API kavramlarını özetler. Detaylar uygulama ilerledikçe genişletilecektir.

## Çekirdek Bileşenler
- Plugin Manager
- Module Loader
- Security Manager
- Config Manager
- Reporting Engine
- Update Service
- Integration Bus

## CLI Komutları (Plan)
- `kb --eklenti-listesi`
- `kb --eklenti-yukle &lt;modul_adi&gt;`
- `kb --eklenti-kaldir &lt;modul_adi&gt;`
- `kb --eklenti-guncelle &lt;modul_adi&gt;`
- `kb --eklenti-bilgi &lt;modul_adi&gt;`
- `kb --modul-olustur &lt;modul_adi&gt;`
- `kb --modul-test &lt;modul_adi&gt;`
- `kb --modul-dogrula &lt;modul_adi&gt;`
