# Katkıda Bulunma

Topluluk katkıları memnuniyetle karşılanır. Lütfen aşağıdaki süreci takip edin.

## Kod Standartları
- Tutarlı stil ve isimlendirme
- Testler ve dokümantasyon

## Test Gereksinimleri
- Unit ve smoke testler
- Bağımlılık taraması

## Dokümantasyon Şablonları
- Modül açıklaması, kullanım örnekleri ve izinler

## Güvenlik Denetimi
- İmza doğrulama ve bağımlılık güvenliği

## Pull Request Prosesi
1. Fork oluşturun
2. Dal açın (özellik/onarım)
3. Testleri çalıştırın
4. Dokümantasyonu güncelleyin
5. PR gönderin
