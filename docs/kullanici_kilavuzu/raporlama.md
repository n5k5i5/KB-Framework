# Raporlama

Standart rapor yapısı aşağıdaki bölümlerden oluşur:
- `rapor_meta`: zaman, sürüm, hedef(ler), kullanılan modüller
- `bulgular`: modül bazlı içerik
- `istatistikler`: toplam/kritik bulgu, tarama süresi
- `gizlilik`: maskeleme ve veri sınıflandırma

## Çıktı Formatları
- JSON (varsayılan)
- XML
- HTML
- PDF
- Markdown

## Arşivleme
- Varsayılan çıktı klasörü: `raporlar/`
- Versiyon ve tarih bazlı isimlendirme önerilir
