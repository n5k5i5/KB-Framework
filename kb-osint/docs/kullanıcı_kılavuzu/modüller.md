# Modüller

Kategorilere göre modül listeleri ve kullanım örnekleri.

## Komutlar

- Tüm modüller:
  ```
  modul_liste
  ```

- Detaylı:
  ```
  modul_liste detayli
  ```

- Kategoriye göre:
  ```
  modul_liste kategori <kategori_adi>
  ```
  Örnek kategoriler: `domain`, `ip`, `email`, `sosyal_medya`, `kisi`, `kurumsal`, `gelismis_arama`, `analiz`, `gorsel`, `raporlama`, `turkiye`, `api`, `araclar`, `guvenlik`, `protokoller`, `topluluk_entegrasyon`

- Kategorileri göster:
  ```
  modul_kategoriler
  ```

- Arama:
  ```
  modul_ara <metin>
  ```

- Aktif/Pasif:
  ```
  modul_liste aktif
  modul_liste pasif
  ```

## Örnek

```
KB-OSINT MODÜL LİSTESİ
Toplam 50+ modül yüklü
--------------------------------------------------------------------------------
 1. kb_domain_whois
     Açıklama: Domain WHOIS bilgilerini toplar.
     Versiyon: 0.1.0
     Kategori: domain
     Yazar: KB-OSINT Team
```
